Pengenalan HTML

HTML atau singkatan dari Hypertext Markup Language, adalah sebuah bahasa markup yang berfungsi untuk memberitahu sebuah browser untuk menampilkan sebuah konten. HTML pertama kali diciptakan oleh Tim Berners-Lee yang sekarang menjabat sebagai direktur organisasi World Wide Web Consortium.

Dalam sebuah website, HTML merupakan sebuah manekin yang dapat dihias dengan CSS sebagai aksesorisnya dan dengan bahasa pemrograman Javascript, manekin tadi dapat diberi kemampuan untuk berinteraksi.
